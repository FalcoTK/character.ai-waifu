# 📚 CAI-WAIFU
Simple way to cai api, using PYCAY lib, and EasyGoogleTranslate

# 🔨 How To Use:
- Download Uisng Git
```
git clone https://github.com/FalcoTK/Cai-Terminal.git
```
- Install
```
pip install -r requirements.txt
```
- Run
```
Python Main.py
```
